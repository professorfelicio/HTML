    Aula 10! - Links ou Hiperligações

    Como criar Hyperligações ou Links

  • O que é um Hyperlink ou Hiperligação?
    • - Falar do conceito base do link, diferenciando Hiperlink de URL.
    Ancoras
  • Links Externos
    • - Utilizar tag <a>
    escrever a e carregar no ENTER
    Irá aparecer o href dentro da tag aqui terá de se colocar o URL completo incluindo o http
    Se queremos que o nosso site não seja fechado utilizar target e rel 
    
        <a href="" target="_blank" rel="external"> Este é o texto do link </a>
    Se queremos que a página no link não seja relacionada com a nossa página para o algoritmo utilizar o
    rel="nofollow"
  • Links Internos
    • - Criar uma nova página HTML na pasta do site. - Utilizar tag <a>
    Dentro do href e das "" carregar em CTRL+SPACE para aparecer as páginas HTML na pasta.
    Não precisa de ter o http://
    No rel utilizar next 
    
        <a href="" rel="next"> Este é o texto do link interno </a>

    Para um link numa página que não é a página principal do nosso site utilizar o
    rel="prev"
    Podemos colocar páginas dentro de pastas
    Para apontar para ficheiros em pastas anteriores utilizar o ../ no href 
    
    <a href="../index.html" rel="prev"> Este é o texto do link para a página inicial</a>

                <h2 class="destaque">

    Pseudo Classes
    - Pseudo classes são utilizadas para eventos dos vários elementos de uma página
    - As Pseudo Classes são separadas de : do elemento.
    - Estas são algumas das pseudo classes de CSS3:

    :active
    :checked
    :disabled
    :empty
    :enabled
    :fullscreen
    :focus
    :hover
    :invalid
    :link
    :read-write
    :required
    :target
    :valid
    :visited

    Lista Completa de Pseudo Classes

    - Exemplo:
    - Utilizar pseudo classes no style.css ou no style do head 
    
                h1:hover{
                    background-color: yellow;
                    text-shadow: 1px 1px 2px black;
                }

    Pseudo Elementos
    - Pseudo elementos são utilizados para modificar elementos visuais a parte de elementos de uma página
    - Os Pseudo elementos são separados de :: do elemento
    - Estss são alguns dos pseudo elementos de CSS3:

    ::after
    ::before
    ::first-letter
    ::first-line
    ::marker
    ::selection

    Lista Completa de Pseudo Elementos

    - Exemplo:
    - Utilizar pseudo elements no style.css ou no style do head 
    
            p::first-letter{
                color: white;
                background-color: black;
                font-weight: bold;
                font-size: 20px;
            }

    Filhos
    - Se queremos referenciar um elemento dentro de um outro elemento dizemos que ele é filho.
    - Para referirmos um filho colocamos o elemento pai depois o > e depois o elemento filho.
    - Exemplo o h1 do header é diferente de o h1 do main por isso podemos referir o h1 do header como: 

    
            header > h1 {
                color: blue;
            }

    Aula 37! - CSS GRID!

    O que é CSS Grid
    - CSS Grid é uma forma esquemática de desenhar páginas de web HTML+CSS utilizando um formato de grelha
    - O CSS Grid utiliza um conceito de grelha estilo tabela para criar o esquema visual de uma página
    Conceitos Básicos de Grid Layout

    - Podemos utilizar o MS Word para exemplificar este conceito
    Exemplo em Word

    - Iniciamos por criar os vários containers e items em HTML
    - Os containers são zonas da página que vão possuir os vários items do conteudo da página 

    
        <main> {
            <section class="grid1"> {
                <aside class="cx c1"> Caixa 1 </aside>
                <aside class="cx c2"> Caixa 2 </aside> 
                <aside class="cx c3"> Caixa 3 </aside> 
                <aside class="cx c4"> Caixa 4 </aside> 
                <aside class="cx c5> Caixa 5 </aside> 
                <aside class="cx c6> Caixa 6 </aside> 
                <aside class="cx c7> Caixa 7 </aside> 
                <aside class="cx c8"> Caixa 8 </aside> 
                <aside class="cx c9"> Caixa 9 </aside> 
                <aside class="cx c10"> Caixa 10 </aside> 
                <aside class="cx c10"> Caixa 10 </aside> 
                <aside class="cx c11"> Caixa 11 </aside> 
            }
            </section>
        }
        </main>
    - Isto vai fazer um container chamado grid1 e vamos colocar items do tipo cx e sub tipo c1...c10
    - Estes tipos são dados por classes a os items vão ter duas classes cx e c1 a c10

    Declarar um CSS Grid
    - Depois de construir-mos o container vamos configurar o container e os items no css
    - Primeiro teremos de dizer que o grid1 é do tipo grid
    - Depois vamos ter de dizer qual é a altura do container utilizando a medida "vh" ou "viewport height", sendo 100vh, 100% da altura total da página.
    - Depois podemos dizer quantas colunas e linhas estão na grelha de items dentro do contentor utilizando o grid-template-columns e grid-template-rows assim como o tamanho das mesmas
    
        .grid1{
            display: grid;
            height:100vh;
            grid-template-columns: 70px 300px 100px;
            grid-template-rows: 150px 500px;

        }
    - Com a medida px ou pixels fazem items com tamanhos fixos
    - Como existem mais items que linhas as restantes linhas vão ter um tamanho igualmente distribuido de acordo com o resto do viewport
    - Podemos fazer uma página mais responsiva utilizando medidas relativas como % 
    
        .grid1{
            display: grid;
            height:100vh;
            grid-template-columns: 15% 50% 35%;
            grid-template-rows: 10% 60% 5%;

        }
    - Novamente existem mais items que linhas os restantes items vão ter o restante da viewport height
    - Também pode ser utilizado o auto para distribuir tudo automaticamente 
    
            grid-template-columns: auto auto auto;
            grid-template-rows: auto auto;
    Profundidade dos items
    - Além das 2 dimensões os elementos Grid também têm a 3a dimensão utilizando o z-index:
    - Começamos por diminuir uma das colunas para cortar o conteudo do item 
    
        grid-template-columns: auto 30px auto auto;
    - Depois vamos modificar o z-index de um dos items cortados para vir para a frente 
    
        .c2{
            background-color:darkorange;
            z-index: 1;
        }
    - Quanto maior for o z-index mais para a frente será puxado o conteudo de um item.
    - Podemos colocar itemos por cima de outros e não só conteudo
    - Modificar o tamanho da caixa 2 ela fica por cima das restantes 
    
        .c2{
            background-color:darkorange;
            height: 200px;
            widht: 200px;
            z-index: 1;
        }

    Modificar local de um elemento
    - Continuamos com a organização dos elementos da grid
    - Podemos escolher em que lugar na grid um elemento irá ficar se o quiser-mos tirar da ordem, utilizando o grid-column e o grid-row
    - Vamos mudar a caixa 8 para o lugar da caixa 1. 

    
        .cx8 {
            grid-column: 1/2;
            grid-row: 1/2;
        }
    - Isto significa que a caixa 8 irá:
    • Desde a primeira linha da grelha até à segunda linha da grelha na horizontal
    • Desde a primeira linha da grela até à segunda linha da grelha na vertical
    - Isto também permite modificar o tamanho da caixa na grelha 
    
        .cx8{
            grid-column: 1/4;
            grid-row: 1/3;
        }
    Utilizar o span
    - Também podemos utilizar o atributo span na grid-column e na grid-row 
    
        .c7{
            background-color:burlywood;
            grid-column: span 3;
        }
    - Isto irá fazer com que a caixa 7 "cubra" 3 colunas

    Medida FR
    - No Grid Layout (e Flexbox) é muito utilizada a medida fr que é uma fração
    - O tamanho das colunas e das linhas poderão ter a medida em fr 
    
        .grid1 {
            grid-template-columns: 1fr 1fr 1fr;
        }
    - Isto significa que a grelha irá ter 3 colunas de tamanhos iguais e responsivas.
    - Coluna 1, 2 e 3 irão cada uma ter uma fração igual da grelha.
    - Podemos utilizar valores de fr diferentes 
    
        .grid1 {
            grid-template-columns: 1fr 2fr 1fr;
        }
    - Isto significa que a Coluna 1 irá ocupar 1 fração da grelha, a Coluna 2 2 frações e a coluna 3 1 fração. - Podemos também utilizar para as linhas 
    
        .grid1 {
            grid-template-rows: 1fr 1fr 1fr;
        }

    Aula 39! - CSS GRID!

    Utilização do repeat
    - Para facilitar a repetição de elementos em linhas e colunas podemos utilizar o repeat (nº repetições, tamanhos)

    
        .grid1{
            grid-template-columns: repeat (3, 1fr);
        }
    - Este código irá dar 3 colunas de 1 fr cada.
    - Podemos alternar tamanhos das colunas utilizando mais que um valor nos tamanhos 
    
        .grid1{
          
            display: grid;
            height:50vh;
            grid-template-columns: repeat(3,1fr 10px);
        }
    Gap entre linhas e colunas
    - Um Grid Gap é um espaço entre os elementos da grelha
    - Podemos fazer espaços entre colunas com grid-column-gap: e entre as linhas com grid-row-gap: 
    
        .grid1{
          
            grid-column-gap: 20px;
            grid-row-gap: 10px
        }
    - Também podemos declarar espaços entre colunas e linhas numa só linha com grid-gap: 
    
        .grid1{
          
            grid-gap: 20px 10px;
            
        }
